# CarUNI - Ride Sharing App

This project was developed based on a prototype I created during university.

The main goal was to transform that idea into a functional web application while improving my skills in software development and test automation.

## Purpose

This project was built to:

- Practice fullstack development (frontend + backend)
- Apply authentication concepts (JWT)
- Work with databases (MongoDB)
- Create a system suitable for automated testing
- Improve code organization and problem-solving skills

## Technologies Used

- Node.js
- Express
- MongoDB
- JavaScript (Vanilla)
- HTML & CSS

## Features

- User registration and login
- Authentication using JWT
- Create rides
- View available rides
- Responsive user interface

## Testing Focus

The application was designed with testing in mind, allowing validation of:

- Authentication flow
- API endpoints
- UI interactions
- Data persistence
